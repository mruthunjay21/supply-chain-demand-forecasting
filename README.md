Project Overview

This project builds an end-to-end demand forecasting and inventory planning system using historical retail sales data.

Key components include:

Data ingestion using SQL

Data cleaning and preprocessing using Python

Exploratory data analysis and visualization

Time-series demand forecasting using Prophet

Inventory optimization using EOQ and reorder point models

Business Insights

Identified seasonal sales trends

Forecasted demand for next 90 days

Estimated optimal inventory ordering quantity

Calculated reorder points to prevent stockouts
